Game Instructions

  • Select a difficulty level and theme to start the game.
  • Each game consists of 8 rounds. Solve the anagram related to your chosen theme in each round.
  • Time to find the word depends on the difficulty level: 45 seconds for Normal and 20 seconds for Hard.
  • Correct answers earn you 3 points.
  • Need help? You can ask for a hint, but it will deduct 1 point in Normal and 2 points in Hard.
