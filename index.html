Anagram Master

Will you be the next Anagram Master?

Each game consists of 8 rounds. In each of them you are presented with an anagram related to the theme you have selected and have to find the right word.

The time you have to find the word in each round depend on the difficulty level you select. In the Normal level there are 60 seconds and in the hard only 30.

Every correct answer gives you 3 points.

You can also ask for a hint for each word. If you use this option you will only get 1 point if you can then find the solution.
